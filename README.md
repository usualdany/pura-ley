# 📋 PURA LEY - Plataforma Legislativa Transparente de Costa Rica

## ¿Qué es PURA LEY?

**PURA LEY** es una plataforma **neutral, transparente y descentralizada** que facilita el acceso público a información sobre proyectos legislativos de Costa Rica.

### Nuestro propósito

Permitir que los ciudadanos costarricenses:
- 🔍 **Busquen** proyectos de ley fácilmente
- 📊 **Visualicen** el estado y avance legislativo
- 📚 **Aprendan** sobre el proceso legislativo
- 🌍 **Acceda** a información pública de forma transparente

### ¿Qué NO es PURA LEY?

❌ **NO** permite participación política directa  
❌ **NO** permite votación ciudadana  
❌ **NO** recopila datos personales  
❌ **NO** tiene afiliación política  

Esto garantiza **neutralidad y seguridad**.

## 🚀 Características

✅ **Buscador avanzado** de proyectos de ley

## 📋 Cómo usar

### 1. Clonar el repositorio
```bash
git clone https://github.com/usualdany/pura-ley.git
cd pura-ley
```

### 2. Abrir en el navegador
Simplemente abre `index.html` en tu navegador o usa un servidor local:

```bash
# Si tienes Python 3
python -m http.server 8000

# Si tienes Node.js con http-server
npx http-server
```

Luego abre `http://localhost:8000` en tu navegador.

---

## 🔧 Cómo agregar más proyectos

Edita el archivo `script.js` y agrega objetos al array `proyectosData`:

```javascript
{
    id: 13,
    expediente: "25,999",
    titulo: "Ley de ejemplo",
    tema: "Educación",
    estado: "En Comisión",
    impacto: "Alto",
    fecha: "2025-06-01"
}
```
